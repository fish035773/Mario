# 遊戲網址 https://assignment2-bf0fd.web.app

# START MENU
    1. 直接按START可不登入開始遊戲，分數不會被紀錄
    2. 可輸入信箱與密碼註冊 / 登入，通關後分數將被登記在排行榜
    3. LEADERBOARD按鈕可瀏覽玩家分數紀錄

# SELECT LEVEL
    1. 按LEVEL 1開始遊戲

# GAME
    1. Player
        # 方向鍵 / AD鍵移動，SPACE跳躍
        # 掉出地圖邊界或碰到敵人將掉一顆心，玩家會重生且分數清零。三顆心全掉即是GAMEOVER。
    2. Enemy: 撞牆轉向，除食人花以外的怪皆可踩頭殺死
    3. Coins: 有50元及10元面值的錢幣，吃到後分數會增加相對應的面值
    4. 問號方塊: 頂開問號方塊吃蘑菇可使體型增大，具備一次性抵擋敵人傷害的保護效果
    5. 終點有Luigi / Peach / 彩虹薯餅三種物件，導到三種結局CG，碰到任一種物件即通關

# ANIMATION
    1. 玩家有走路 & 跳躍動畫
    2. 三種怪有三種不同的動畫

# MUSIC
    1. STARTMENU音樂
    2. 關卡音樂
    3. 吃到金幣將播放音效
    4. 吃到問號方塊將播放音效
    5. 玩家跳躍音效
    6. 玩家死亡音效
    7. 玩家體型增大音效
    8. 玩家體型縮小音效

# UI
    1. 左上三顆愛心代表玩家三條命，命用完即是GAMEOVER
    2. 右上分數吃錢幣增加
    3. 中間三分鐘計時器，時間用完玩家未通關即是GAMEOVER

# APPEARENCE
    使用自繪素材，包含:

    1. 環境物件：問號方塊、水管、彩虹薯餅通關物件、遊戲背景、Mario Title
    2. UI 素材：錢幣/愛心 UI Icon、排行榜外框、輸入框、按鈕組件、遊戲標題
    3. 結局 CG：針對三種结局（Peach、Luigi、彩虹薯餅）繪製了結局 CG 圖

# BONUS
    1. 登入登出功能
    2. 排行榜登記各用戶的分數
    3. Deployed to Firebase
